üéØ What are Feature Maps?

Feature Maps are the outputs produced when convolutional filters are applied to input data. Each filter detects specific patterns or features in the image, creating a 2D array that shows where and how strongly those features appear.

Layer 1 Features

Edges, Lines, Corners

Layer 2-3 Features

Shapes, Textures, Patterns

Deep Layer Features

Objects, Complex Patterns

üëÅÔ∏è What is Receptive Field?

Receptive Field is the region of the input that influences a particular neuron's output. As we go deeper into the network, neurons can "see" larger areas of the original input through the cascade of previous layers.

Layer 1: Local Receptive Field (3√ó3)
Layer 2: Expanded Receptive Field (5√ó5)
Layer 3: Large Receptive Field (7√ó7)

üß† Key Insight:

Notice how a single neuron in deeper layers is influenced by a much larger area of the original input. This hierarchical processing allows CNNs to understand complex patterns and relationships.

üîó How They Work Together

Feature Maps and Receptive Fields work in harmony:

  • Early Layers: Small receptive fields detect simple features (edges, corners)
  • Middle Layers: Medium receptive fields combine simple features into shapes and textures
  • Deep Layers: Large receptive fields recognize complex objects and patterns
