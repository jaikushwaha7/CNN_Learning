📚 Understanding the Concepts

What happens in a CNN layer?

1. Convolution (Linear Operation): A kernel (filter) slides across the input image, computing weighted sums of pixel neighborhoods. This detects features like edges, textures, and patterns.

2. ReLU Activation (Non-linear): Applies the function max(0, x) to introduce non-linearity, creating sparsity and feature selectivity.

3. Combined Effect: Linear convolution preserves spatial relationships while ReLU creates selective, sparse feature representations.

🔍 Deep Dive: How ReLU Affects Image Features

Linearity in Convolution

Convolution is a linear operation that preserves proportional relationships. If you double the input intensity, the output doubles too. This maintains the spatial structure and allows complex feature combinations.

ReLU Non-linearity Impact

ReLU introduces thresholding behavior:

  • Sparsity: 40-60% of values become zero, creating sparse representations
  • Selectivity: Only strong feature responses survive
  • Gradient Flow: Prevents vanishing gradients during training
  • Computational Efficiency: Zero values require no computation in subsequent layers

Visual Effects on Images

Early CNN layers with ReLU create high-contrast feature maps where edges, corners, and textures are either strongly present (bright) or absent (dark). This builds the foundation for hierarchical feature detection in deeper layers.
