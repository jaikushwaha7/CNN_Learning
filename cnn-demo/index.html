üéØ What are Multiple Channels in CNNs?

In Convolutional Neural Networks, channels represent different feature maps or dimensions of data. Think of channels as different "views" or "perspectives" of the same information, allowing the network to process and learn diverse patterns simultaneously.

Key Concept: Multiple channels enable CNNs to extract rich, hierarchical features from input data, progressively building from simple edges to complex objects.

üíª Code Example: PyTorch Implementation

import torch
import torch.nn as nn

class MultiChannelCNN(nn.Module):
    def __init__(self):
        super(MultiChannelCNN, self).__init__()
        
        # Layer 1: 3 input channels ‚Üí 32 output channels
        self.conv1 = nn.Conv2d(in_channels=3, out_channels=32, 
                              kernel_size=3, padding=1)
        
        # Layer 2: 32 input channels ‚Üí 64 output channels  
        self.conv2 = nn.Conv2d(in_channels=32, out_channels=64,
                              kernel_size=3, padding=1)
        
        # Layer 3: 64 input channels ‚Üí 128 output channels
        self.conv3 = nn.Conv2d(in_channels=64, out_channels=128,
                              kernel_size=3, padding=1)
        
        self.pool = nn.MaxPool2d(2, 2)
        self.relu = nn.ReLU()
    
    def forward(self, x):
        # x shape: [batch_size, 3, 224, 224] (RGB input)
        
        x = self.relu(self.conv1(x))  # [batch_size, 32, 224, 224]
        x = self.pool(x)              # [batch_size, 32, 112, 112]
        
        x = self.relu(self.conv2(x))  # [batch_size, 64, 112, 112]
        x = self.pool(x)              # [batch_size, 64, 56, 56]
        
        x = self.relu(self.conv3(x))  # [batch_size, 128, 56, 56]
        x = self.pool(x)              # [batch_size, 128, 28, 28]
        
        return x
